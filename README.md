# App-Books
## To get started with this project you need to create a directory to host the project
## Open Git Bash
## Clone into your directory
Git clone: `git clone https://github.com/hoangvuHY/App-Books.git`
## Path to the App-Books file
`cd App-Books`
## Install packages
`npm install`
## Project start
`npm start`
